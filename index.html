01Digital Twin Platform Bentley iTwin · Full-Stack
A full-stack enterprise Digital Twin platform built on Bentley iTwin, with a React + TypeScript frontend, Flask/Python microservice backend, and AWS RDS PostgreSQL database. Provides live 3D model integration, IoT monitoring, construction progress tracking, and quality certification — all in one connected system.
Monitoring
Asset Performance
Three role-based dashboards (Executive, Operations, Engineering) with KPIs, health scoring, and trend analysis. The 3D model is color-coded by asset health status. Selecting an asset in the table highlights it in the 3D model, and vice versa.
Workflow
Facility Service Requests
End-to-end maintenance request lifecycle: submission, admin assignment, technician execution, and sign-off. Requests can be raised by clicking directly on the 3D model. Automated email notifications at every workflow stage.
Real-Time
IoT Monitoring & Control
Live sensor data (temperature, pressure, alarms) displayed within the 3D environment. Room-based device management with visual alerts, color-coded severity, and historical trend views.
4D BIM
Construction Progress Tracking
IWP progress table with bulk update and undo/redo. 3D model color-coded by completion status. Includes 4D animated timeline simulation, before/after comparison, and embedded Power BI analytics linked to the 3D model.
Quality
QC & Certification
Automated intake from the Progress module — completed IWPs are forwarded for QA/QC review automatically. Certificate creation, inspection tracking, and 3D colorization distinguishing certified vs. uncertified elements.
BIM Data
Hierarchy Visualization
8-level drill-down navigation: Project → Zone → Area → Unit → Discipline → Category → Element. Custom color mapping applied to the 3D model in real time — handles 118,000+ elements efficiently.
Analytics
Power BI Dashboards
Embedded Microsoft Power BI reports authenticated via Azure Active Directory — enterprise-grade analytics with bidirectional filtering between charts and the 3D model.
Admin
Administration Panel
Centralized platform management for user roles, multi-tenant configuration, and system settings — built with a modular sidebar architecture.
Automation & Backend Pipelines
IoT Data Simulation Automated pipeline that generates realistic sensor data, pushes it to the database, and updates live Power BI dashboards in real time — supports single-run and continuous modes.
Room Data Extraction Reads room coordinates from the 3D model and maps them to the database, enabling accurate IoT device positioning and spatial data linkage.
Progress → QC Handoff Automated cross-module workflow: completed IWPs are forwarded from Progress Tracking to QC & Certification for review — no manual handoff required.
Automated Notifications Event-driven email service integrated into the service request workflow — triggers on creation, assignment, and completion stages.
Database Provisioning Setup pipelines that auto-provision the full database schema, roles, and configurations from scratch — with version migration support.
Data Seeding Pipelines that populate realistic test data for asset performance and operations — enabling rapid demo environment setup.
Health Monitoring Backend tooling for database health checks, schema auditing, multi-tenant validation, and data integrity analysis.
02Smart Demolition Viewer Autodesk Forge (APS) · Full-Stack
A web-based platform built on Autodesk Platform Services (APS / Forge), enabling intelligent management, visualization, and analysis of demolition projects directly in the browser — integrating 3D BIM viewing with material logistics, circular economy analytics, and permit management.
3D Viewing
Viewer & Demolition Sequence
Dual 3D viewer for simultaneous building and site map display. Animated step-by-step demolition sequence linked to an interactive Gantt timeline with phase color-coding.
Analytics
Material Analytics Dashboard
Live charts (type, condition, recyclability, circular economy score, location) synchronized with the 3D model. Clicking a chart highlights the corresponding elements in the viewer in real time.
Visualization
Filtering & Colorization
Multi-criteria filtering and real-time color-coding of model elements — by material type, recyclability, demolition status, and circular economy score.
Logistics
Material Marketplace
Portal for browsing, posting, and requesting construction and demolition materials. Includes personal inventory management, order tracking, and route cost estimation between sites.
BIM Data
3D Material Extractor
Reads and exports material quantity data directly from the BIM model for procurement or inventory reporting.
Compliance
Demolition Permit Management
Digital submission of official demolition permits — capturing applicant details, asbestos certifications, and contractor data. Permits are stored and automatically exported as formatted PDF documents.
Automated Workflows
Model Processing Pipeline Uploading a Revit or CAD file triggers automatic cloud translation via the APS Model Derivative API — making the model viewable in the browser with no manual steps.
Automated PDF Export Permit submissions automatically generate a professionally formatted PDF document for record-keeping and regulatory compliance.
Cross-Module Data Sync Material, order, and marketplace data persists across sessions and can be exported to CSV for sharing or reporting.

Capabilities Delivered

Data Validation Rule-based validation services that check incoming data against defined schemas. Non-compliant records are rejected with specific error feedback before entering the system.
Database API Integration CRUD operations and data synchronization between multiple external sources and cloud databases via REST APIs.
Automated Notifications Event-driven email service that triggers notifications for workflow state changes.
File & Folder Automation Automated file organization, batch renaming, and folder structure management.
Automated Testing Unit and integration tests written for all deployed services — validating correctness, data integrity, and API behavior.
Flow-Based Integration Microservice orchestration using flow-based programming — linking services into automated pipelines for end-to-end BIM data processing workflows.
