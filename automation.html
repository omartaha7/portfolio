01Data Automation & Pipelines
These pipelines were built and deployed within Digital Twin platforms (Bentley iTwin, Autodesk Forge) — the platforms served as the environment where the automation ran, connecting 3D models, databases, IoT sensors, and dashboards.
Pipelines & Data Automation
IoT Data Simulation Automated pipeline that generates realistic sensor data (temperature, pressure, alarms), pushes it to the database, and updates live dashboards in real time — supports single-run and continuous modes.
Room Data Extraction Automated process that reads room coordinates from a 3D model and maps them to the database — enabling accurate IoT device positioning and spatial data linkage without manual input.
Database Provisioning Setup pipelines that auto-provision the full database schema, roles, and configurations from scratch — with version migration support for schema updates.
Data Seeding Pipelines that populate realistic test data for asset performance, maintenance history, and operations — enabling rapid demo and development environment setup.
Health Monitoring Backend tooling for database health checks, schema auditing, multi-tenant validation, and data integrity analysis.
02Workflow & Process Automation
Automations that replaced manual handoffs — notifications, status transitions, and cross-system data flows across different modules and platforms.
Automated Workflows
Progress → QC Handoff When a work package is marked complete, it is automatically forwarded to the QC & Certification system for review — no manual handoff, no missed packages.
Email Notification Service Event-driven email service that triggers on workflow state changes: request creation, assignment, and completion — keeping all stakeholders automatically informed.
Model Processing Pipeline Uploading a model file triggers automatic cloud translation via the Autodesk APS API — making the 3D model viewable in the browser with no manual conversion steps.
Document Generation Form submissions automatically generate formatted PDF documents — removing manual document creation from the workflow entirely.
PowerBI Data Integration Automated script that populates simulated IoT data, pushes it to the database, and automatically updates live PowerBI dashboards embedded in the application — supports both static and live database connections.
